<h1 align="center">Extended Regex</h1>
Utilizzare il comando egrep per risolvere i seguenti esercizi (dando esempi di possibili input):<br>
- Matchare sequenze alfabetiche di almeno un carattere, sia maiuscole che minuscole <br>
- Togliere tutte le righe di commento di uno script Bash matchando le righe che iniziano con '#' <br>
- Matchare in un file le righe che contengono sequenze di 'a' ripetute da 3 a 6 volte <br>
- Matchare in un file tutte le righe che contengono una delle seguenti parole: apple, pear, orange
