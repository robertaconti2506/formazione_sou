<h1 align="center">Numero di processsi</h1>
Contare il numero totale di processi attualmente in esecuzione sulla propria macchina senza utilizzare utility interattive come "top". <br><br>

Ho realizzato uno script bash che conta il numero di processi attualmente in esecuzione sulla macchina. <br> Ho utilizzato il comando "`ps`" per ottenere la lista dei processi attivi e "`wc -l`" per contarli (`-l` conta le righe).
