<h1 align="center">Odd/Even</h1>
Scrivere uno script che accetti un argomento numerico e stampi la sequenza di numeri da 1 a quel numero con l'indicazione se è pari o dispari. Controllare il corretto numero di argomenti passati allo script e che l'argomento sia di tipo numerico, in caso contrario generare una condizione di errore. <br><br>

Il seguente script bash analizza l'input dato da linea di comando e, se rispetta la richiesta (numero intero maggiore di 0), stampa a video tutti i numeri da 1 al numero inserito come argomento, aggiungendo se si tratta di un numero pari o dispari. 

Per verificare lo script: 
```bash
./even_odd <argomento>
```
